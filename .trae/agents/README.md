# Agency Agents (Local Install)

This repo can install the “agency-agents” collection into:

`/.trae/agents/agency-agents`

## Install / Update

Run:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\install-agency-agents.ps1
```

## How To Use (in this repo)

- Browse the installed agent files under `.trae/agents/agency-agents/`.
- Copy the agent’s “Identity / Mission / Workflows” text into your prompt when you want to activate that persona.
